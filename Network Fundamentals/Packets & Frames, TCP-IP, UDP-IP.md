- A frame is at layer 2 - the data link layer, meaning there is no such information as IP addresses. 
- when we are talking about anything IP addresses, we are talking about packets.
- When the encapsulating information is stripped away, we're talking about the frame itself.
 
TCP/IP:

- The TCP/IP protocol consists of four layers:
    - Application
    - Transport
    - Internet
    - Network Interface
-  information is added to each layer of the TCP model as the piece of data (or packet) traverses it. As you may recall, this process is known as encapsulation - where the reverse of this process is decapsulation.
 
Ports:

- ports are a numerical value between **0** and **65535** (65,535).
- Any port that is within **0** and **1024** (1,024) is known as a common port.
- FTP 21, SSH 22, HTTP 80, HTTPS 443, SMB 445, RDP 3389
