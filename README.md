# TryHackMe: Pre-Security Pathway Notes

This repository contains my personal documentation, notes, and practical lab write-ups for the **Pre-Security** pathway on TryHackMe. This curriculum covers the foundational technical concepts required to break into cybersecurity, focusing on networks, operating systems (Linux.Windows), and web applications.

---

## 🗺️ Pathway Roadmap & Modules

The repository is structured day-by-day corresponding to the timeline of core modules completed:

| Step | Module | Description |
| :--- | :--- | :--- |
| **01** | **Intro to CyberSecurity** | Foundational concepts, career paths, and basics of defensive/offensive security. |
| **02** | **Network Fundamentals** | Deep dive into OSI layers, packets, IP addressing, routing, LANs, and core protocols. |
| **03** | **Web Fundamentals: How The Web Works** | Understanding HTTP/HTTPS, DNS, cookies, web servers, and browser interactions. |
| **04** | **Linux Fundamentals** | Hands-on with the terminal, file permissions, directory structures, and shell commands. |
| **05** | **Windows Fundamentals** | Exploring the OS internals, Event Viewer, registry, and fundamental administrative tools. |
